🧠 Why This Actually Works

Most tools guess which terminal to focus. We don't.

ApproachAccuracy
Process tree walking (mintty → bash → claude)✖ Broken on MSYS2
"First mintty wins"✖ Wrong window with multiple terminals
Window title heuristics⚠ ~80% at best
GetForegroundWindow at hook time → direct HWND✅ 100%

The key insight: When Claude Code fires a permission prompt, your active window IS the correct terminal. We capture the HWND at that exact millisecond and embed it in the notification button. No guessing. No heuristics. Pointer-level precision.
